# 01_fuentes

Fuentes y referencias base del proyecto.

Categorías:

- `producto/`: definiciones de producto, pricing, mercado, feedback.
- `arquitectura/`: diagramas, decisiones técnicas, ADRs externas.

- `qa/`: evidencias QA históricas, screenshots, reportes.
- `negocio/`: documentos comerciales, marca, billing, legal.

No guardar secretos aquí.
