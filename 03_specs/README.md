# 03_specs

Specs de trabajo para ONAI OCR.

Cada spec debe incluir:

- tarea(s) relacionadas;
- objetivo;
- alcance incluido/excluido;
- archivos o módulos relevantes;
- criterios de aceptación;
- verificación mínima;
- riesgos;
- reporte esperado del agente.

Los specs son contratos de trabajo: si cambia el alcance, actualizar el spec antes de seguir.
