# 04_trabajo

Workbench por tarea.

Reglas:

- Cada tarea usa carpeta `TXX_nombre/`.
- Guardar aquí notas, borradores, diagnósticos, scripts en desarrollo, reportes intermedios.
- No guardar secretos.
- No guardar entregables finales aquí: mover/copiar a `05_entregables/`.
- Cada carpeta TXX debe tener README si el trabajo dura más de una sesión.
